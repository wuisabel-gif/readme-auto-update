Skill and Action

Review the evidence and the proposed README interactively in Claude Code or Codex - no model API key required - or run it unattended on a schedule with the GitHub Action.

Evidence, not guesswork

Reads owned projects, organization work, open-source contributions (forks included, credited to their upstream), and private activity - then writes only what the repositories support.

Privacy-safe by default

Private repository names, links, and descriptions stay hidden; private work appears only as one anonymized total unless you explicitly opt in.

Your voice, kept

Updates only a marked, managed section - your intro and footer are never touched. Write with AI (OpenAI or Anthropic) or with deterministic, zero-cost rules.
